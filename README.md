# URL Shortener

A simple web app to shorten long URLs into easy-to-share short links using Python and Flask.

## Features
- Shortens any long URL into a 5-character short link
- Redirects short link to the original URL
- Lightweight and beginner-friendly
- Can be extended to a full-scale URL shortening service

## Tech Stack
- Python
- Flask
- HTML (minimal form)
